Applied

Real-World Scenarios

How the framework maps to actual business workflows.

🏥 Healthcare — Clinical Document Processing Tier 3 → DeepSeek V3.2
High volume, structured extraction from clinical docs. Doesn't need creative reasoning — needs reliable JSON output at scale. $0.25/M input vs $5/M on a frontier model = 20x savings. Reserve Tier 1 for complex appeals or edge cases.
📊 Sales — Pitch Decks & Competitive Strategy Tier 1 → Opus 4.6
Persuasive writing, competitive positioning, nuanced messaging. This is where frontier models justify their cost. A bad pitch loses the deal. Worth paying $25/M output.
📧 Operations — Email Triage & Drafting Tier 2 → Kimi K2.5
Classify importance, draft replies, summarize threads. Quality 89 is more than enough for email. $0.45/M vs $3/M on Sonnet. Save ~85% for the same job.
🤖 DevOps — Always-On Background Agents Tier 4 → Qwen 3.5-9B (local)
Triage checks, tool routing, simple classification running 24/7. API models cost $1–5/M per call. Local model = $0 forever, no API dependency, sub-100ms response. Runs on a $600 Mac Mini.
🌐 Content — User-Facing Q&A Widget Tier 3 → Gemini 2.5 Flash
User-facing Q&A needs to be good but also cheap at scale. Gemini has a free tier (1K req/day) and 2M context for loading reference docs. Free up to ~30K queries/month.

Decision Tree

Quick Decision Flowchart

Start here when choosing a model.

Q1: Does the data need to stay on-device?
→ Yes → Tier 4 Local Qwen 3.5-9B or Llama 4 Scout
Q2: Is being wrong expensive? (legal, medical, financial, strategy)
→ Yes → Tier 1 Frontier Opus 4.6 or GPT-5.2 Pro
Q3: Is it creative, nuanced, or requires good writing?
→ Yes → Tier 2 Performance Sonnet 4.6, GPT-5.2, or Kimi K2.5
Q4: Is it high-volume or structured? (parsing, classification, bulk processing)
→ Yes → Tier 3 Value DeepSeek V3.2, GPT-5 Nano, or Gemini Flash
Default: Start with Tier 2 — optimize down once you know your quality floor.
Pro move: Layer your tiers. Use Tier 3/4 for triage → route complex cases to Tier 1/2. You get frontier quality where it matters and value pricing for the 80% that doesn't need it.

The pattern: Frontier models shine on judgment tasks (persuasion, strategy, nuance). Value and local models are equally good at mechanical tasks (parsing, classification, extraction). Most teams overspend by using frontier models for mechanical work.

Summary

Key Takeaways

🏆 Frontier is for thinking

Strategy, persuasion, complex reasoning, high-stakes output. Use Opus 4.6 or GPT-5.2 Pro. Don't waste these on classification or parsing.

⚡ Chinese labs are disrupting Tier 2

GLM-5 (quality 94, $0.95), Kimi K2.5 (89, $0.45), DeepSeek V3.2 (79, $0.25). These aren't budget models — they're genuine competitors at 5-10x lower price.

🏠 Local models are finally useful

Qwen 3.5-9B has best-in-class tool calling (beats 30B and 80B models). Runs at 50 t/s on a $600 Mac Mini for $0. The agent-on-device era is here.

📐 Layer your tiers

Don't pick one model for everything. Use cheap models for triage, mid-tier for daily work, frontier for critical tasks. A tiered system saves 60-90% vs using frontier for everything.
